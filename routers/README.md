# routers/

HTTP-слой приложения. Определяют эндпоинты, инжектируют сессию БД и сервис через `Depends`.

Все роутеры регистрируются в `main.py` через `app.include_router(...)`.

---

## books.py — `/api/v1/books`

| Метод | URL | Статус | Описание |
|-------|-----|--------|----------|
| GET | `/` | 200 | Список всех книг |
| GET | `/{book_id}` | 200 / 404 | Книга по id |
| POST | `/` | 201 / 404 | Создать книгу |
| PUT | `/{book_id}` | 200 / 404 | Обновить книгу |
| DELETE | `/{book_id}` | 204 / 404 | Удалить книгу |
