# services/

Слой бизнес-логики. Вызывают репозиторий, валидируют данные, бросают `HTTPException` при ошибках, возвращают DTO.

---

## BookService (`book.py`)

Зависимость: `BookRepository`.

| Метод | Описание |
|-------|----------|
| `get_all()` | Возвращает список всех книг |
| `get_by_id(book_id)` | Возвращает книгу, 404 если не найдена |
| `create(data, db)` | Проверяет существование всех переданных авторов, создаёт книгу |
| `update(book_id, data, db)` | Обновляет книгу, 404 если не найдена |
| `delete(book_id)` | Удаляет книгу, 404 если не найдена |

Вспомогательный метод `_get_authors(db, author_ids)` — получает список авторов по id, бросает 404 если хотя бы один не найден.
